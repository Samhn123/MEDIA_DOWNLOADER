# MEDIA_DOWNLOADER
A simple yet powerful web-based Media Downloader that allows users to download videos and audio from platforms like YouTube, Instagram, and more using a clean and responsive interface. A modern and powerful Media Downloader Web App that allows users to download videos and audio from various platforms in multiple formats and qualities.

✨ Features
🎬 Download videos in 360p, 720p, 1080p
🎵 Convert videos to MP3 audio
⚡ Fast downloading using yt-dlp
📊 Smooth download handling
📱 Fully responsive UI (Mobile + Desktop)
🎨 Clean and user-friendly interface

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Tools: yt-dlp, FFmpeg
